**Stock Management System**
**Overview**
This is a Stock Management System designed to help businesses track inventory, monitor stock levels, and manage product data efficiently.
The system allows users to log in, view stock, and manage inventory through a simple interface.

**Features**
User authentication (login system)
View available stock
Track low stock items
Dashboard for inventory overview
Basic backend integration with database

**Technologies Used**
HTML
CSS
JavaScript
Node.js
XAMPP (Apache & MySQL)
